# More Cool Stuff 

- [DIVE](https://urol-e5.github.io/deep-dive-genome-browser/) - Data-driven Insights into Variation and Epigenetics across coral species. This data delivery platform spotlights a genome browser to explore non-coding RNA features across three species.
- [oystergen.es](https://oystergen.es) Ostra chilensis: Epigénoma Análisis en Net) is an integrated web platform dedicated to advancing research on Ostrea chilensis, the Chilean flat oyster.
- [Coral MOSAiC Project](https://urol-e5.github.io/MOSAiC/) - This portal serves complementary purposes of direct data delivery and offering a multi-species genome browser. This project encompasses a time-series study of three coral species— Acropora pulchra, Pocillopora tuahiniensis, and Porites evermanni —sampled from reef sites in Mo’orea, French Polynesia.
- [Contra](https://github.com/sr320/ConTra) Framework for identifying context-dependent regulatory interactions in multi-omics data
- [VIMS Resazurin Trial Shiny app](https://sr320.shinyapps.io/resazurin-explorer/)
